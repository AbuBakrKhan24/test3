Root route

Users route

So in the URL in your web-browser it should look like the one that is below, this is the user route, that route's webpage will give you information in an JSON format of our users information.

These are not links but just how th URL should look, so all you adding after the end of the URL is "/users" to the Root route's URL

http://localhost:6969/users

So in the URL in your web-browser it should look like the one that is below, this is the products route, that route's webpage will give you information in an JSON format of our products information.

These are not links but just how th URL should look, so all you adding after the end of the URL is "/products" to the Root route's URL
